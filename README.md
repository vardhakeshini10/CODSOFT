# Movie Genre Classification

## About the Project
In this project, I built a machine learning model that predicts the genre of a movie based on its description.

## Dataset
The dataset contains movie details like ID, title, genre, and description. I used the description column as input and genre as output.

## What I Did
- Loaded the dataset using pandas
- Cleaned the data by removing missing values
- Converted text into numerical form using TF-IDF
- Split the data into training and testing sets
- Trained the model using Logistic Regression

## Result
The model was able to predict movie genres with an accuracy of around 58%.

## Example
Input: "A hero fights villains to save the world"  
Output: Action

## Conclusion
This project helped me understand how text classification works using machine learning. I learned how to preprocess text data and train a model to make predictions.
