# Credit Card Fraud Detection

## About the Project
In this project, I built a machine learning model to detect whether a credit card transaction is fraudulent or not.

## Dataset
The dataset contains transaction details such as transaction amount, location, city population, and time. It also includes a column indicating whether the transaction is fraud or not.

## What I Did
- Loaded and explored a large dataset using pandas
- Checked the distribution of fraud and normal transactions
- Selected important numerical features to avoid memory issues
- Split the dataset into training and testing sets
- Trained the model using Random Forest algorithm

## Result
The model was able to classify transactions with high accuracy.

## Conclusion
This project helped me understand how fraud detection works in real-world scenarios and how to handle large datasets efficiently using machine learning.
